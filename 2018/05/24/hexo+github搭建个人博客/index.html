环境配置

node环境

  • Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。
  • Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。
  • Node.js 的包管理器 npm，是全球最大的开源库生态系统。
    浏览器打开node官网下载8.11.2版本的msi双击安装

hexo项目托管到github

本机验证github的key

  1. 打开github 注册个人账号
    并记录下用域名和邮箱，并且验证邮箱

  2. 桌面右键打开git bash here

接下来依次打开：c:/Users/Administrator/.ssh路径
找到：id_rsa.pub文件右键选择记事本打开，全选并复制里面的内容(key)

  1. 回到github页面，找到头像右侧倒三角下拉菜单，选择settings
  2. 在页面左侧选择ssh and gpk
  3. 点击右上角绿色 add ssh按钮
  4. 在表单输入title(随意)和key(粘贴之前复制到的key)
    然后点击绿色create ssh按钮
  5. 回到刚才创建key的命令行窗口验证用户名和邮箱

如果验证成功，没有任何提示。
